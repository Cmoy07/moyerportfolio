Web-scrapper & Data Interpreter

A simple Python program that scraps data and then interprets it.

Description

My Web-scrapper program is a tool built solely on Python. This program grabs, or "scraps", Co2 emissions data off of worldometers.info, then interprets it into graphs and tables that contextualize the data and make it understandable. Build as a final project for my programming course, to showcase my skills and proficiencies in Python. This project helped me learn a plethora of Python libraries, such as: Pandas, MatPlotLib, Numpy, Requests, BeautifulSoup, and SQLite3.
